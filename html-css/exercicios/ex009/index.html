Código-fonte

O comando document.getElementById('teste') é escrito em JavaScript, e foi formatado utilizando a tag code.

O comando open('a','fatura-mensal.txt') é escrito em Python e também foi formatado utilizando a tag code.

Embora a tag formate o trecho do código corretamente, há um problema com a estrutura do código, por exemplo:

def fazerCopiaArquivo(nomeArquivo): nomeCopiaArquivo = nomeArquivo + "-copia.txt" copiaArquivo = open('a', nomeCopiaArquivo) arquivo = open('r', nomeArquivo) for linha in arquivo.readlines(): copiaArquivo.write(linha) print("Arquivo copiado com sucesso!\n")

O código foi formatado para estar na mesma linha e perdeu sua indentação. E, para evitar isto, utiliza-se a tag pre antes da tag code. Segue abaixo: 

        
    def fazerCopiaArquivo(nomeArquivo):
        nomeCopiaArquivo = nomeArquivo + "-copia.txt"
        copiaArquivo = open('a', nomeCopiaArquivo)
        arquivo = open('r', nomeArquivo)

        for linha in arquivo.readlines():
            copiaArquivo.write(linha)

        print("Arquivo copiado com sucesso!\n")
