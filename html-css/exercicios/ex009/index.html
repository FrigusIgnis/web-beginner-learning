Listas Ordenadas

A lista ordenada é uma das facilidades encontradas no HTML5, a qual utiliza a tag ol para gerar uma lista por ordem, geralmente, ordinal(Seus itens são separados e categorizados pela tag li).

Listas não ordenadas

A lista não ordenada é uma lista onde a ordem não tem importância, seguindo um padrão aleatório utilizando a tag ul para gerar a lista.

Listas mistas

A lista mista consiste no uso das duas listas anteriores em conjunto.

Listas de definição

A lista de definição consiste, como o próprio nome já diz, definir um elemento dentro da lista, a qual utiliza a tag dl para gerar a lista e as tags dt e dd para atribuir o elemento e sua definição, respectivamente.
