CSS inline

O uso de estilização inline é compreendido no uso do atributo style numa tag para tal feito. Como exemplo, foi alterado a cor de fundo da página, utilizando o atributo em questão na tag <body>

CSS interno/local

O uso de CSS interno consiste em criar a tag <style> no head para construir seletores e atribuir elementos de estilização no HTML.
